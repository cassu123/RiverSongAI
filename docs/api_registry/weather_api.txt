SERVICE: Weather API (OpenWeatherMap)
WHAT IT DOES: Gives River Song current weather conditions and a 5-day forecast for any city, including temperature, conditions, and daily highs and lows.
HOW TO GET IT:
  1. Go to openweathermap.org/api and click "Sign Up" (or "Sign In" if you already have an account).
  2. After signing in, click your username in the top-right corner and select "My API Keys".
  3. Copy the default API key shown, or click "Generate" to create a new one.
  4. Paste the key as the value for WEATHER_API_KEY in your .env file.
  5. Set DEFAULT_LOCATION to your city in "City,CountryCode" format (e.g. Chicago,US or London,GB).
  6. Set WEATHER_UNITS to imperial for Fahrenheit or metric for Celsius.
  Link: https://home.openweathermap.org/api_keys
VARIABLE NAME: WEATHER_API_KEY, DEFAULT_LOCATION, WEATHER_UNITS
WHERE TO PUT IT: .env
NOTES:
  - The free tier includes 1,000 API calls per day and access to the current weather and 5-day/3-hour forecast endpoints. Both are all River Song uses.
  - New API keys can take up to 2 hours to activate after creation. If you get a 401 error, wait and try again.
  - The 5-day forecast returns data in 3-hour intervals. River Song filters and summarizes these for day-of-week queries ("what's the weather Friday").
  - WEATHER_UNITS only affects the display unit -- the API returns whichever unit you choose, so pick one and leave it.
