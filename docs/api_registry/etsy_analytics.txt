SERVICE: Etsy Open API v3
WHAT IT DOES: Lets River Song pull your Etsy shop orders, revenue, listing views, and favorites from your seller dashboard.
HOW TO GET IT:
  1. Go to developers.etsy.com and sign in with your Etsy seller account.
  2. Click "Create a New App".
  3. Fill in the app name (e.g. "River Song"), a brief description, and select "Personal Use"
     or "I'm building for myself" as the use type.
  4. Under "What type of access do you need?", select "Read access for shop data".
  5. Copy the API Key (also called Keystring) shown after creation.
     This goes in ETSY_API_KEY.
  6. Set the redirect URI to: http://localhost:8000/callback/etsy
  7. Use the OAuth 2.0 PKCE flow River Song generates to authorize your Etsy shop.
     River Song will store the resulting access and refresh tokens automatically.
     The access token goes in ETSY_ACCESS_TOKEN.
  8. Find your Shop ID:
     Log in to etsy.com/your/shops/me/dashboard. Your shop name appears in the URL.
     Or call: GET https://openapi.etsy.com/v3/application/users/me?api_key={ETSY_API_KEY}
     and read the "shop_id" from the response.
     This goes in ETSY_SHOP_ID.
  Link: https://developers.etsy.com/documentation
VARIABLE NAME: ETSY_API_KEY, ETSY_ACCESS_TOKEN, ETSY_SHOP_ID
WHERE TO PUT IT: .env
NOTES:
  - Etsy's API v3 uses OAuth 2.0 with PKCE. There is no "client secret" -- the flow is
    key + PKCE challenge only.
  - Access tokens expire after 3600 seconds (1 hour). Refresh tokens are valid for 90 days.
    River Song will need to implement automatic refresh once the login flow is built.
  - The Etsy API has a rate limit of 10 requests per second and 10,000 requests per day per key.
    River Song's on-demand dashboard calls will not come close to this.
  - Order revenue from the API is in the shop's native currency. River Song displays
    the currency as returned by the API.
  - "Views" in the analytics refers to listing views, not shop visits. Shop-level visit
    stats require separate calls to the Shop Stats endpoint.
  - Etsy does not require app review for personal seller access. Your key is active immediately.
