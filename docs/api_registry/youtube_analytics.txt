SERVICE: YouTube Data API v3 + YouTube Analytics API (Google)
WHAT IT DOES: Lets River Song pull your YouTube channel subscribers, total views, watch hours, and estimated revenue from YouTube Studio analytics.
HOW TO GET IT:
  1. Go to console.cloud.google.com and sign in with the Google account that owns your channel.
  2. Create a new project (or use an existing one). Name it "River Song" or similar.
  3. In the left menu: APIs & Services -> Library.
     Search for and ENABLE both:
       - "YouTube Data API v3"
       - "YouTube Analytics API"
  4. Go to APIs & Services -> OAuth consent screen.
     Choose "External" (works for personal use). Fill in app name and your email.
     Add these scopes: https://www.googleapis.com/auth/youtube.readonly
                       https://www.googleapis.com/auth/yt-analytics.readonly
     Add your Google account as a Test User.
  5. Go to APIs & Services -> Credentials -> Create Credentials -> OAuth 2.0 Client ID.
     Choose "Web application". Set the authorized redirect URI to:
       http://localhost:8000/callback/youtube
     Copy the Client ID and Client Secret.
     These go in YOUTUBE_CLIENT_ID and YOUTUBE_CLIENT_SECRET.
  6. Use the OAuth flow River Song generates to authorize your YouTube account.
     River Song will store the access and refresh tokens automatically.
     The refresh token (long-lived) goes in YOUTUBE_REFRESH_TOKEN.
  7. Find your Channel ID:
     youtube.com -> click your profile picture -> Your Channel -> copy the ID from the URL.
     (format: UC...) This goes in YOUTUBE_CHANNEL_ID.
  Link: https://developers.google.com/youtube/v3
VARIABLE NAME: YOUTUBE_CLIENT_ID, YOUTUBE_CLIENT_SECRET, YOUTUBE_REFRESH_TOKEN, YOUTUBE_CHANNEL_ID
WHERE TO PUT IT: .env
NOTES:
  - YouTube API uses Google OAuth 2.0 (same system as google_oauth.txt). If you already have
    a Google OAuth app set up for River Song, you can enable the YouTube APIs on that same
    project instead of creating a new one. Add the scopes and create a separate OAuth client.
  - The YouTube Analytics API requires the channel to be in good standing and have at least
    some historical data. Brand new channels with no views may return empty results.
  - Revenue (RPM, estimated earnings) requires YouTube Partner Program membership
    (monetization enabled). If your channel is not monetized, those fields return 0.
  - Refresh tokens for YouTube/Google do not expire unless revoked. River Song uses the
    refresh token to get new short-lived access tokens automatically.
  - The free quota for YouTube Data API v3 is 10,000 units per day. River Song's read-only
    dashboard calls use very few units and will not approach this limit under normal use.
